# chat-adapter-state-ioredis

ioredis-style (pipeline-oriented) Redis state adapter for chat-py

## Install

```bash
pip install chat-adapter-state-ioredis
# or with uv
uv add chat-adapter-state-ioredis
```

> Part of the [chat-py](https://github.com/desplega-ai/chat-py) workspace — a Python port of [vercel/chat](https://github.com/vercel/chat). See the root README for motivation, architecture, and cross-package usage.
